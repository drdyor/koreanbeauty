# Envy Theme Design Enhancements for K-Beauty Glow

## Key Design Elements from Envy Theme

### 1. **Layout & Structure**
- Breadcrumb navigation on product pages
- Flexible product image layouts (small/medium/large with adjustable widths)
- Thumbnail positioning options (left, below)
- Image zoom functionality on product pages
- Sale badges prominently displayed

### 2. **Typography System**
- Multiple serif and sans-serif font options
- Clean, elegant typography hierarchy
- Professional font pairings

### 3. **Product Display**
- Image galleries with thumbnail navigation
- Side thumbnails for product images
- Image swap on hover
- Prominent sale badges
- Clean product information layout

### 4. **Navigation**
- Mobile-friendly slide-out menu
- Multi-level dropdown support
- Integrated search in mobile menu
- Customer account links in navigation

### 5. **Color & Theme Variables**
- Customizable color scheme system
- Header/Footer color controls
- Menu background/text colors
- Sale label colors
