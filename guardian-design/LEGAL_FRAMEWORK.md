**What We're Using**:

1. **Silva Method Countdown (5-4-3-2-1)**
   - Technique developed in 1960s
   - No active trademark on the countdown itself
   - We can say: "Based on Silva Method techniques"

2. **Gateway Breathing (5-5-5)**
   - Monroe Institute technique
   - Breathing patterns are not copyrightable
   - We can say: "Inspired by Gateway Experience breathing"

3. **Lozanov Superlearning (4-4-4-4 + Baroque)**
   - Suggestopedia principles from 1960s-70s
   - Public domain methodology
   - We can say: "Using Lozanov-style rhythmic suggestion"

4. **Polyvagal Exercises**
   - Based on Stephen Porges' theory (published science)
   - Techniques like humming, orienting are not owned
   - We can say: "Polyvagal-informed nervous system regulation"

**Bottom Line**:
- ‚úÖ Techniques = public domain (we can use freely)
- ‚úÖ Names = attribute sources ("based on", "inspired by")
- ‚ùå Don't claim we invented these methods
- ‚ùå Don't claim medical efficacy without FDA approval
